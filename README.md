# ctd-transcription-analysis
Analysis code for the study "A Tunable CTD Grammar Governs the Spatial Programming of the Transcription Cycle".
## Repository contents

- `traveling_ratio_go_analysis.R`  
  R script for traveling ratio calculation, statistical comparison, gene grouping, and GO enrichment analysis.

- `Readthrough_index_analysis.R`  
  R script for readthrough index calculation, statistical comparison, and visualization.

- `TTseq_MAplot.R`  
  R script for differential expression analysis and custom MA plot generation from TT-seq data.
